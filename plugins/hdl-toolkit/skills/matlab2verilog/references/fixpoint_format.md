# 定点数格式规范

## S(W,F) 格式定义

```
S(W,F) = W位有符号定点数，F位小数

W = 总位宽 (包含符号位)
F = 小数位宽
I = W - F - 1 = 整数位宽 (不含符号位)
```

## 格式示例

| 格式 | 符号 | 整数 | 小数 | 范围 | 精度 |
|------|------|------|------|------|------|
| S(8,7) | 1 | 0 | 7 | [-1, 1) | 0.0078 |
| S(14,11) | 1 | 2 | 11 | [-4, 4) | 0.00049 |
| S(16,15) | 1 | 0 | 15 | [-1, 1) | 0.000031 |
| S(18,17) | 1 | 0 | 17 | [-1, 1) | 0.0000076 |
| S(24,23) | 1 | 0 | 23 | [-1, 1) | 0.00000012 |

## 位宽计算规则

### 加法

```
S(W,F) + S(W,F) = S(W+1, F)

示例: S(14,11) + S(14,11) = S(15,11)
```

### 减法

```
S(W,F) - S(W,F) = S(W+1, F)

示例: S(14,11) - S(14,11) = S(15,11)
```

### 乘法

```
S(W1,F1) × S(W2,F2) = S(W1+W2, F1+F2)

示例: S(14,11) × S(14,11) = S(28,22)
```

### N 个数累加

```
S(W,F) + S(W,F) + ... (N个) = S(W+ceil(log2(N)), F)

示例: 8个 S(14,11) 累加 = S(14+3,11) = S(17,11)
```

### 截断/舍入

```
S(W,F) → S(W',F')  其中 F' < F

方法: 右移 (F-F') 位 + 舍入/截断
```

## 量化公式

### 浮点转定点

```python
fixed_val = round(float_val * (2 ** F))
```

### 定点转浮点

```python
float_val = fixed_val / (2 ** F)
```

## Verilog 声明

```verilog
// S(14,11) 格式
parameter DATA_WL  = 14;  // 总位宽
parameter DATA_DWL = 11;  // 小数位宽

reg signed [DATA_WL-1:0] data;  // [13:0]
```

## 与其他格式对比

| 格式 | S(W,F) 等价 | 说明 |
|------|-------------|------|
| Q15 | S(16,15) | TI DSP 标准 |
| Q2.11 | S(14,11) | Qm.n 格式 (m不含符号) |
| Q3.11 | S(14,11) | Qm.n 格式 (m含符号) |
| Fix<14,11> | S(14,11) | Xilinx 格式 |

## 常用定点格式

### 数据通道

| 应用 | 推荐格式 | 说明 |
|------|----------|------|
| ADC 输出 | S(12,11) ~ S(16,15) | 根据 ADC 位数 |
| 滤波器数据 | S(14,11) ~ S(18,15) | 留余量防溢出 |
| FFT 数据 | S(16,15) ~ S(18,17) | 考虑级数增长 |

### 系数

| 应用 | 推荐格式 | 说明 |
|------|----------|------|
| FIR 系数 | S(12,11) ~ S(18,17) | 根据精度需求 |
| 窗函数 | S(16,15) ~ S(18,17) | 需要较高精度 |
| 旋转因子 | S(16,15) ~ S(18,17) | sin/cos 表 |
